# Wiyak Content Assistant

> AI content creation assistant for Arabic and English

## ✨ Features

- Bilingual content
- Blog post generator
- Social captions
- Translation support
- Tone adjustment
- SEO optimization

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn
- API keys (Google AI, OpenAI, etc.)

### Installation
```bash
git clone https://github.com/ZOOKA-AI/wiyak-content-assistant.git
cd wiyak-content-assistant
npm install
```

### Configuration
Create `.env` file:
```env
API_KEY=your_api_key_here
```

### Usage
```bash
npm start
```
